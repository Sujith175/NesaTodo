## MERN Todo List Application

This is a simple todo list application built with MERN stack:

- **M**ongoDB: MongoDB atlas database for storing todos.
- **E**xpress.js: Node.js web framework for the backend.
- **R**eact: JavaScript library for building the frontend.
- **N**ode.js: Runtime environment for running the server.

**Prerequisites**

- Node.js and npm installed on your system. You can download and install Node.js from the official website: https://nodejs.org/
-  MongoDB cloud service

**Installation**

1. **Clone the repository:**
   ```bash
   git clone <repository_url>

2. Navigate to the project directory:


3. Install server-side dependencies:
    cd server
    npm install

4. Install client-side dependencies: 
    cd client
    npm install

5.Setup Environment Variables    
    MONGO_URI=your_mongodb_connection_string

6.Start the server:
    cd server
    npm start

7.Start the client:
    cd client
    npm start
